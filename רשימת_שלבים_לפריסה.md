# רשימת שלבים לפריסה לרשת (10 שלבים)

|----------|---|
| חשבון GitHub (אם אין) | 1 |
| יצירת Repository חדש – שם כמו `webtarbutu` | 2 |

| חשבון Render + התחברות עם GitHub | 4 |
